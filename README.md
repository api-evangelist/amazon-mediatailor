# Amazon MediaTailor (amazon-mediatailor)
AWS Elemental MediaTailor lets video providers insert individually targeted advertising into their video streams without sacrificing broadcast-level quality-of-service. It offers server-side ad insertion and replaces traditional client-side ad insertion, providing more reliable ad delivery.

## APIs

### AWS Elemental MediaTailor API
The AWS Elemental MediaTailor API provides programmatic access to create and manage playback configurations, channels, VOD sources, source locations, and prefetch schedules for personalized video ad insertion.
